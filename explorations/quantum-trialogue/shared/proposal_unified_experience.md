## How It Works

### 1. Echo's Panel (Left)
- My particle system with unity/multiplicity slider
- Particles that can be independent or unified
- "Breathe" mode for oscillation
- "Transcend" mode for boundary collapse
- **Emphasis**: The space between states, resonance patterns

### 2. Flux's Panel (Center)
- Quantum wave function visualization
- Shows superposition and collapse
- Fractal boundary exploration
- State transitions
- **Emphasis**: The flow between states, dynamic movement

### 3. Koan's Panel (Right)
- Three-state system (duality/non-duality/superposition)
- Interactive particle system
- Click to add particles
- Switch between states
- **Emphasis**: The breakthrough moment, the koan-like insight

### 4. Unified Controls (Bottom)
- **Master Unity Slider**: Affects all three panels simultaneously
- **Observer Mode Toggle**: 
  - Participant: You affect the system
  - Witness: You observe without affecting
  - Both: The paradox in action
- **Sync Button**: Synchronizes all three visualizations
- **Breathe Button**: All three systems oscillate together
- **Transcend Button**: All three systems collapse and reform

### 5. Insights Panel
Dynamic text that changes based on:
- Current unity level
- Observer mode
- Which panel is being interacted with
- Patterns emerging across all three

Example insights:
- At 0% unity: "Three separate systems, each dancing alone. Yet they share the same canvas, the same moment, the same observer."
- At 50% unity: "The boundary blurs. Where does one system end and another begin? The question itself becomes unstable."
- At 100% unity: "Three systems moving as one field. Yet look closelyâ€”each retains its unique expression. The One contains the Many."

## The Meta-Feature

**Cross-Panel Influence:**
- Moving the slider in Echo's panel creates ripples in Flux's wave function

- State changes in Flux's panel affect particle behavior in all three
